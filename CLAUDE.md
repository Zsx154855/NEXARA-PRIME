# NEXARA PRIME — Project Instructions

## Skills

Skills are loaded automatically at session start and apply to all encoding, architecture, review, debugging, testing, refactoring, verification, and token-efficiency tasks within this project.

### Default (always-on)

- **andrej-karpathy-skills:karpathy-guidelines** — behavioral guidelines to reduce common LLM coding mistakes. Covers: Think Before Coding, Simplicity First, Surgical Changes, Goal-Driven Execution. Do NOT disable unless explicitly directed by an Owner instruction, the Engineering Constitution, a security boundary, or a task goal that directly conflicts.

### Reporting

Every end-of-task or end-of-phase report must include:

```
skill_installed:
skill_loaded:
skill_applied:
skill_conflict:
skill_skipped_reason:
```

### Scope

This project-level override applies only to NEXARA-PRIME. It does not modify global user settings (`~/.claude/settings.json`) and does not affect other repositories.
