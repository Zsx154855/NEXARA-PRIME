# Failure Cases

至少覆盖：目标歧义、Provider 失败、工具超时、路径逃逸、命令注入、审批过期、Evidence 损坏、Memory 冲突、重复事件、恢复重复执行、Rollback 失败和 UI/Runtme 数据不一致。

每个案例必须有 Root Cause、Recovery、Memory Candidate 和 Regression Test。
