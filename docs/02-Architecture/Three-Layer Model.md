# Three-Layer Model

## Layer 1 — Human Decision Core

人类拥有目标设定、价值判断、最终审批、暂停、接管、撤权、安全模式和回滚权。人类不是普通节点，而是治理主体。

## Layer 2 — Hermes + Eight Personas

Hermes 只负责接收、路由、观察、升级和恢复；具体专业能力由运行时职能和工具承担。

| Persona | Primary role |
|---|---|
| Atlas | 安全、风险、权限、治理 |
| Nyx | 数据分析、洞察、模式发现 |
| Orion | 研究、检索、证据采集 |
| Solace | 协调、依赖、协作 |
| Vertex | 工程、推理、技术执行 |
| Echo | 沟通、表达、交付 |
| Lumen | 解释、教学、知识转化 |
| Kairos | 评测、优化、成本、效率 |

人格是用户可理解的表达层；Orchestrator、Planner、Reviewer 等是可动态分配的运行职能层。
